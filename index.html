How to Play

  1. Choose a puzzle size and select a puzzle number.
  2. Each row and column shows a regex pattern — fill every cell with a single character so the full row/column string matches the regex.
  3. Click Check to validate. Green cells match their row and column regexes; red cells don't.
  4. Stuck? Use Hint to reveal one cell (costs score points).
  5. Solve the puzzle as fast as you can with the fewest hints for the best score!

Frequently Asked Questions

Do I need to know regex to play?

No! The built-in puzzles start simple. The cheat sheet above teaches you the basics as you go. Each puzzle helps you learn common regex patterns naturally.

How is the score calculated?

Base score is 1000 points. Points are deducted based on solve time (−2 per second) and hints used (−100 per hint). Minimum score is 0.

Can I create my own puzzles?

Not yet — but 10 handcrafted puzzles across three grid sizes give you plenty to practice with. More puzzles and a custom puzzle editor are planned for future updates.
